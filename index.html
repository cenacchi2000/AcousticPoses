/ 02 — proof-reel results

Paper results and supporting evidence

PCK is the fraction of frames whose normalized landmark error is below 10%. Event F1 measures whether recovered motion peaks occur at the same moments as the video-reference landmarks.

Full benchmark metrics are computed over 78 landmark proof clips and the main paper’s 4,639 held-out motion-proxy test clips. Same-clip external avatar-SOTA overlays are not shown because those methods have not been exported into this landmark target space.

Code and datasets used

This anonymous page includes the main local scripts used to build the benchmark, run the landmark proof, validate MediaPipe proxies, and generate the manuscript-facing figures. Dataset downloads remain with the original public dataset providers.

Benchmark data

MELD + CREMA-D + RAVDESS

Experiments use 21,454 public real audio-video clips: 11,132 MELD, 7,442 CREMA-D, and 2,880 RAVDESS clips.

Speaker-disjoint split: 13,657 train, 3,158 validation, and 4,639 test clips.

Visual proof data

Held-out landmark subset

The visual proof videos use held-out clips from the same public datasets. MediaPipe video-reference landmarks provide the evaluation target; the orange reconstruction is generated from audio-derived features only.

The checked landmark proof set contains 78 held-out clips with 147 dense 3D facial landmarks per frame.

Dataset access

Original providers

  • MELD: multimodal emotion lines from TV dialogue.
  • CREMA-D: acted emotional speech audio-video clips.
  • RAVDESS: acted speech and song audio-video clips.

Raw videos are not redistributed here; scripts expect locally downloaded dataset folders.
